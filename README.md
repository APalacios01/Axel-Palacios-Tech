# Axel Portfolio (Grey/Black/White + Midnight Purple)

This starter matches the layout vibe you showed (big hero text + portrait frame) but uses your colors:
**grey, white, black, and midnight purple**.

## Files to edit
- `index.html` (main layout)
- `assets/css/styles.css` (colors + layout)
- `assets/img/portrait.svg` (replace with your photo)
- `assets/resume/resume.pdf` (optional)

## Publish on GitHub Pages
### Best option (personal site)
1. Create a repo named: **YOUR-USERNAME.github.io**
2. Upload everything in this folder to the repo root
3. Repo → **Settings → Pages**
4. Source: **Deploy from a branch**
5. Branch: `main`, Folder: `/ (root)`

## Replace the portrait
Replace `assets/img/portrait.svg` with a real image:
- Use `portrait.jpg` or `portrait.png`
- Then update `index.html` to point to it (search for `assets/img/portrait.svg`)
